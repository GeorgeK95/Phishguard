# PhishGuard expanded ontology notes

This branch expands the ontology so it is comparable in size to the provided pizza ontology example.

Current ontology size:
- OWL ontologies: 1
- Classes: 121
- Object properties: 15
- Data properties: 10
- Named individuals in JSON model: 62

The expansion is domain-driven, not random boilerplate. It adds structured cybersecurity concepts for threats, indicators, communication channels, target organizations, attack goals, evidence, risk levels and recommendations.
